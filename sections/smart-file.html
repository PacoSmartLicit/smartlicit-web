Smart File / Contingencias

Contingencias

Errores recuperables y siguientes pasos.

CAPTCHA / anti-botPortal bloquea automatización simple.

Siguiente: cambiar estrategia, horarios, caching o modo asistido.

Login requeridoAcceso completo solo para partes.

Siguiente: modo con credenciales seguras o solo boletín público.

Datos incompletosFaltan campos obligatorios para el portal.

Siguiente: pedir campos adicionales y reintentar validación.

No hay matchExpediente no encontrado.

Siguiente: revisar instancia, órgano y sugerir búsqueda asistida.

Portal caídoError 5xx o timeout.

Siguiente: reintentos con backoff + aviso al usuario.

Conector cambióHTML o estructura distinta.

Siguiente: conector en mantenimiento y fallback a revisión manual.
