Overview

El documento vive en Drive y se trabaja dentro de SmartLicit.

La interfaz no debe sentirse como chat ni como editor genérico. Debe funcionar como una cabina documental guiada: izquierda para orientación, centro para edición y derecha para asistencia contextual.

Regla principal

Drive real

SmartLicit no copia archivos finales desde una carpeta oculta. Los Docs y Sheets nacen y se editan directamente en la carpeta del usuario.

IDs preservados

ID

Cada archivo conserva google_file_id, google_folder_id, última modificación, versión operativa y estado en SmartLicit.
